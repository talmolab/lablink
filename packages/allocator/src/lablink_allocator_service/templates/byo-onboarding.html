BYO Client Onboarding

Run this command on a Linux GPU box to register it as a manual client with this allocator. The command is pre-populated with this deployment's URL and current bootstrap token. 

lablink client register \
  --allocator-url {{ allocator_url }} \
  --register-token {{ register_token }}{% if show_insecure %} \
  --insecure{% endif %}
Note: the register token rotates on every allocator restart. If a BYO operator's lablink client register fails with an auth error, refresh this page to copy the current token.

Prerequisites on the BYO box: Docker + nvidia-container-toolkit (for GPU), and the lablink-cli Python package installed.

    get_host_access(hostname) ¶

    Return (instance_id, public_ip, ssh_key_path) for hostname.

    Any component may be None if unavailable (no credentials, not reachable, etc.). Providers that cannot recover hosts should return (None, None, None) — _reboot_vm gates on can_recover_hosts before calling this, so it will only ever be invoked on capable providers.

    ProvisioningNotSupported ¶

    Bases: Exception

    Raised by a provider whose structural model does not support a lifecycle operation — distinct from ProviderActionNotWired (deferred wiring). For example, ManualProvider cannot ever provision hosts because the operator brings them; raising this signals "never" rather than "not yet".
